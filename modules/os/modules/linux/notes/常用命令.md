#### 获取指定主机的 IP

```bash
ip=$(host "$(hostname)" | awk 'NR==1 {print $NF}')
```

##### 案例

```bash
hosts=(host1 host2 host3)

ip=$(host "$(hostname)" | awk 'NR==1 {print $NF}')

for host in "${hosts[@]}"; do
if [[ "$(host "${host}" | awk 'NR==1 {print $NF}')" != "${ip}" ]]; then
  echo "host=${host}"
fi
done
```
