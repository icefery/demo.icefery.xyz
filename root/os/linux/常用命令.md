#### 获取指定主机的 IP

```bash
ip=$(host "$(hostname)" | awk 'NR==1 {print $NF}')
```

##### 案例

```bash
hosts=(host1 host2 host3)

ip=$(host "$(hostname)" | awk 'NR==1 {print $NF}')

for host in "${hosts[@]}"; do
if [[ "$(host "${host}" | awk 'NR==1 {print $NF}')" != "${ip}" ]]; then
  echo "host=${host}"
fi
done
```

#### 搜索包含有指定内容的所有文件名

```shell
# -r 表示递归搜索子目录
# -l 表示只输出包含匹配内容的文件名，而不显示匹配的具体行内容
grep -rl "abcd" /path/to/directory
```
